# MoFome
Projeto da Disciplina de Projeto de Software
# Funções

## newAccount
Cria conta
## changeProfile
Modifica perfil
## postmanagement
Posta receita
## addFriends
Adiciona amigos
## accountRecovery
Recupera a senha
## addComment
Adiciona comentário
## show
Mostra o perfil
## addAvaliation
Avalia uma receita
## remover_comentario_arquivo
Deleta todos os seus comentários, receitas e tópicos. Não será possível recuperá-los
## topicos
Cria um tópico de discussão
## adicionar_comentario_topico
Adiciona uma resposta a um tópico de discussão
## privacidade
Ativa o modo de privacidade
## privacidade_desativar
Desativa o  modo de privacidade
## RecoverID
Recupera o seu id
